Public Works Assets Maintenance System
======================================

Long name, simple task. Record maintenance data to Garden Grove Public Works assets.

How to run in Docker
====================

    $ docker build -t <name> .
    $ docker run -p 4200:4200 -p 3000:3000 -it <name>
